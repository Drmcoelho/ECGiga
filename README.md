# ECGiga - Curso Completo de ECG

Bem-vindo ao **ECGiga**, o curso mais completo de eletrocardiografia disponível online! 

## 🩺 Sobre o Projeto

O ECGiga é uma plataforma educacional completa para aprender interpretação de ECG, desde conceitos básicos até tópicos avançados. Desenvolvido para estudantes de medicina, enfermeiros, médicos e profissionais da saúde que desejam aprimorar seus conhecimentos em eletrocardiografia.

## 🚀 Funcionalidades

- **6 Módulos Completos**: Do básico ao avançado
  - Fundamentos do ECG
  - Arritmias Cardíacas
  - Isquemia e IAM
  - Hipertrofias
  - Bloqueios
  - Tópicos Avançados

- **Recursos Interativos**:
  - Quiz com questões e explicações detalhadas
  - Casos clínicos práticos
  - Calculadoras ECG (Sokolow-Lyon, QTc)
  - Biblioteca de exemplos de ECG
  - Interface responsiva para dispositivos móveis

- **Design Moderno**:
  - Animação de traçado ECG
  - Interface intuitiva e profissional
  - Navegação suave entre seções
  - Modais interativos com conteúdo detalhado

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 moderno com variáveis CSS e animações
- JavaScript vanilla para interatividade
- Design responsivo (Mobile-first)
- Fontes Google (Inter)
- Font Awesome para ícones

## 📱 Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Navegue pelos módulos clicando em "Estudar Módulo"
3. Pratique com os quizzes interativos
4. Teste conhecimentos com casos clínicos
5. Use as calculadoras ECG para referência

## 🎯 Conteúdo Educacional

### Módulo 1: Fundamentos
- Anatomia do sistema de condução
- Ondas, intervalos e segmentos
- Derivações do ECG
- Frequência e ritmo cardíaco

### Módulo 2: Arritmias
- Ritmo sinusal normal
- Bradicardias e taquicardias
- Fibrilação e flutter atrial
- Taquicardias ventriculares

### Módulo 3: Isquemia e IAM
- Alterações da onda T
- Supra e infradesnivelamento de ST
- Localização do IAM pelas derivações
- Cronologia das alterações

### Módulo 4: Hipertrofias
- Hipertrofia atrial direita e esquerda
- Hipertrofia ventricular direita e esquerda
- Critérios diagnósticos

### Módulo 5: Bloqueios
- Bloqueios atrioventriculares
- Bloqueios de ramo
- Hemibloqueios

### Módulo 6: Tópicos Avançados
- Síndrome de Brugada
- QT longo e curto
- Pré-excitação ventricular
- Marcapassos
- Alterações eletrolíticas

## 📊 Calculadoras Incluídas

- **QTc**: Correção do intervalo QT pela frequência
